Cara menjalankan aplikasi:
Cara 1:
- Buka file index.html pada browser.
Cara 2:
- Buka link berikut https://disk-scheduling-sistem-operasi.w3spaces.com/

Menu Kontrol:
- Input Data: Untuk menambahkan data.
- Reset Data: Untuk mereset data.
- Switch Algoritma: Pilih grafik algoritma mana yang ingin ditampilkan.

Input Data:
- Klik Input Data.
- Masukan posisi head apa input Head Position.
- Masukan data track pertama.
- Klik Tambah Data untuk menambah data track selanjutnya.
- Klik Hapus Data untuk menghapus data track paling akhir.
- Klik Simpan untuk menyimpan data track yang telah diinput.
